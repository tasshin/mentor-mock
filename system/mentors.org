* About This File
This file serves as an index of mentors. Detailed information about each mentor can live in a separate knowledge base (e.g., Obsidian vault, markdown files, etc.).

Each mentor entry includes:
- Type: Real Life, Public Figure, Historical, or Imaginal
- Areas: Their domains of expertise
- Vault: Path to their detailed file (optional)

Types explained:
- *Real Life*: People you know personally
- *Public Figure*: Living people you don't know personally but learn from
- *Historical*: People no longer alive
- *Imaginal*: Fictional characters, archetypes, or composite figures
